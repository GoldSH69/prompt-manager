🐙 GitHub Gist 동기화 안내:

① 토큰 발급:
GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
→ gist 체크 → Generate token

② 사용 방법:
• Push: 현재 프롬프트를 Gist에 업로드
• Pull: Gist에서 프롬프트를 다운로드 (병합)
• 자동 동기화: 프롬프트 추가/수정/삭제 시 자동 Push

③ 다른 기기에서:
같은 Gist ID + 토큰 입력 → Pull → 동일 데이터 복원!
